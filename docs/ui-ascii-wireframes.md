# UI ASCII Wireframes

本文件合并保存当前讨论中整理出的 ASCII 线框。内容用于给项目成员与后续 Copilot session 阅读。

```text name=image-1-multiviewwindow-layouts-v2.txt
OBS Advanced Multiview
Image 1 对应：MultiviewWindow 的实际显示形态

统一术语：
- MultiviewWindow：独立打开的 multiview 窗口。
- cell：基础网格中的单元格，或 span 合并后的显示区域。
- span：多个基础 cell 合并后的区域。
- gutter / border：cell 上下左右之间的间距 / 边缘。
- PGM / PRVW / Scene / Source：第一阶段支持的 OBS 内部信号。
```
